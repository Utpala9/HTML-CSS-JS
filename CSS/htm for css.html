  • display: flex ‚Üí applied to parent
  • Images/videos go inside child boxes
  • object-fit: cover ‚Üí fills box without distortion
  • autoplay loop muted ‚Üí makes video play smoothly

Adding Video to Boxes Built Using display: flex

When boxes are created using display: flex, a video cannot be added using CSS background.

To play a video inside a specific box, the "video" tag must be placed inside the child "div".

Changes Required in HTML

  1. Create a parent "div" and arrange child boxes using display: flex.
  2. Insert the "video" tag inside the required child "div".
  3. Specify width and height attributes in the "video" tag to define the initial size.
  4. Use the "source" tag to link the local .mp4 file.
  5. Add autoplay, muted, and loop attributes for smooth playback.

Changes Required in CSS

  1. Set fixed dimensions for the child boxes.
  2. Use overflow: hidden to keep the video inside the box.
  3. Apply object-fit: cover so the video fills the entire box without gaps.

Importing and Using Custom Fonts in CSS

CSS cannot directly read ZIP files or font links downloaded as

The font file must be extracted and available in a supported format before it can be used in CSS.

Supported font formats for CSS:

  • ."ttf "(TrueType Font)
  • ".otf" (OpenType Font)
  • ".woff"
  • ".woff2" (best and recommended for web)

If the font is not in one of these formats, it should be converted before use.

Steps to import and use a custom font:

  1. Download and extract the font files.
  2. Place the font file inside the project folder (for example, in a fonts/ directory).
  3. Use the @font-face rule in CSS to define the font.
  4. Apply the font using the font-family property.
