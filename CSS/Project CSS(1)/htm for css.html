üé¨ Animated Button Background using CS

This CSS line is used to add a GIF animation as the background of a button and make it look neat and responsive.

  • " background: url("animation.gif") center/cover no-repeat;"
  • url("animation.gif") ‚Üí Sets the GIF as the background image (GIFs loop by default).
  • center ‚Üí Centers the GIF inside the button.
  • cover ‚Üí Scales the GIF to fully cover the button without distortion.
  • no-repeat ‚Üí Prevents the GIF from repeating multiple times

Linking multiple css file in 1 html file

  • One HTML file can be linked with multiple CSS files.
  • All CSS files must be linked inside the "head" tag.
  • If the same style is written in more than one CSS file, the last linked file overrides the others.
  • Using multiple CSS files helps in better organization and readability

display: flex

  • is used to align and arrange items easily in one row or column.

  • display: flex; is applied to the parent element
