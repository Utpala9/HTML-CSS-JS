## What does CSS do exactly?
CSS:
- Changes **colors**, **backgrounds**, and **fonts**
- Controls **spacing** (margin, padding)
- Manages **layout** using Flexbox and Grid
- Handles **alignment and positioning**
- Adds **animations and transitions**
- Makes websites **responsive**

---

## How CSS works (simple)
- CSS targets HTML elements using **selectors**
- Styles are applied based on **priority rules**
- Browser calculates final styles and renders the page

**HTML = Structure**  
**CSS = Design**  
**JavaScript = Functionality**
